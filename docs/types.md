# Leaf basic types

```leaf
// quantum computing specific:
bit, qubit

// additional quantum types:
angle32, angle64, param

// signed integer types:  
i8, i16, i32, i64, i128

// unsigned integer types:
u8, u16, u32, u64, u128

// floating-point types:
f32, f64

// boolean type:
bool

// unit type:
()

```
